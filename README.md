# 多變的天氣互動教材

這是一份國小自然科靜態網頁教材，主題包含：

- 水與天氣的關係
- 天氣圖與天氣變化
- 颱風與防災

## 使用方式

直接開啟 `index.html`，或在本資料夾啟動本機伺服器：

```powershell
python -m http.server 7790
```

再開啟 `http://localhost:7790/`。

## 教材活動

1. 水與天氣：調整時間情境、太陽照射、濕度與高空冷卻，觀察蒸發、成雲、降雨與夜間露水。
2. 天氣圖：點選高氣壓、低氣壓、冷鋒與降雨區，觀察等壓線、環流方向、鋒面推進與雨帶移動。
3. 颱風防災：調整颱風靠近程度與強度，觀察暴風圈、螺旋雨帶、長浪、淹水與山區風險。
4. 天氣小常識：閱讀六個圖文頁籤，搭配 SVG 動畫理解雲、午後雷雨、海陸風、都市熱島、觀雲看天與生活氣象應用。
5. 學習檢核：完成十題形成性評量，每題檢核不同天氣概念。

## 檔案

- `index.html`：教材入口與內容結構
- `styles.css`：RWD 版面、視覺與動畫
- `script.js`：互動模擬、回饋與測驗邏輯
- `teacher-guide.md`：教師使用建議
